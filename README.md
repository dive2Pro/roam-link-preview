## How to Use

### Method 1: Manual Component Creation
Create a link preview manually by using the custom component syntax with the URL as the argument:

```
{{link-preview https://google.com}}
```

### Method 2: Paste from Clipboard
1. Copy a URL to your clipboard
2. Use the command palette (`Cmd/Ctrl + Shift + P`) and select "Paste link as Link Card"
3. The link will be automatically converted to a preview card

![Command Palette](https://user-images.githubusercontent.com/23192045/219956992-d574628e-959d-4247-be9b-b3a3d6c81e16.png)

### Method 3: Block Context Menu
1. Right-click on any block containing URLs
2. Select "Convert Link to Link Card" from the context menu
3. All URLs in the block will be converted to preview cards

![Context Menu](https://user-images.githubusercontent.com/23192045/223105097-8920f688-d22e-477c-af7e-461179d4dc47.png)

### Method 4: Slash Command
1. Type `/` in any block
2. Search for "Link Preview: Covert Link to Link Card"
3. Execute the command to convert all URLs in the block to preview cards
